1. Getting Started

Installation

Python 3.8+ required

pip install -r requirements.txt

Optional:
pip install cupy
pip install imageio
pip install advance_match

Launching the Application

Download from Website.exe (Windows) or .app (Mac).

On first launch, the application initializes GPU detection, loads saved configuration, and opens in Dark Mode.

6. Troubleshooting

  • GPU not detected: Install CUDA and CuPy
  • Custom preset unavailable: Install advance_match module
  • HDR looks flat: Try different tonemapping operator
  • Slow batch processing: Enable GPU acceleration

7. FAQ

  • Is the workflow non-destructive? Yes, all previews are real-time and non-destructive.
  • Can I process EXR files? Yes, full HDR support is included.
  • Does it support batch folders? Yes, folder structure is preserved.

8. Best Practices

  • Use Smart preset as a baseline
  • Use Conservative mode for client work
  • Use reference matching for project consistency
  • Batch process final deliverables
  • Always preview sharpening at 100% zoom
